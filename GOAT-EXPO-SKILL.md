# GOAT Expo Skill (this project)

This app is built with **[GOAT Expo Skill](https://github.com/Abdulla090/goat-expo-skill)** — the production Expo SDK 56 agent skill (navigation, lists, liquid glass, EAS, Sentry, anti–AI-slop).

## Skill in this repo

```
.cursor/skills/goat-expo-skill/
├── SKILL.md
└── references/
```

Cursor agents in this workspace should load `goat-expo-skill` for all React Native / Expo work.

## Install / update globally

```powershell
git clone https://github.com/Abdulla090/goat-expo-skill.git "$env:USERPROFILE\.agents\skills\goat-expo-skill"
```

## Pre-ship verify (this app)
